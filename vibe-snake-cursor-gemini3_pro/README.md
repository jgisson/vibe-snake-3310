# Vibe Snake 3310 - Cursor Gemini 3 Pro edition

A retro Snake game implementation mimicking the classic Nokia 3310 experience.

## Features
- Classic Snake gameplay
- Authentic Nokia 3310 visual style (using photo background)
- On-screen controls (mapped to keypad) + Keyboard support (Arrow keys)
- High score tracking (session based)

## Setup
Ensure `Nokia_3310.png` is present in the root directory for the phone visual.

## How to Play
- Open `index.html` in a web browser.
- Use Arrow keys or click the keypad areas on the phone image to move:
  - 2 key area: Up
  - 8 key area: Down
  - 4 key area: Left
  - 6 key area: Right
  - Center Menu Button: Start/Action
- Eat the food to grow and gain points.
- Avoid hitting the walls or yourself.
